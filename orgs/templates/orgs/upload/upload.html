Upload Process

  1. Select your CSV file
  2. Data will pass through an initial validation stage - you might see warnings when data is missing, but the activity can still load. Or errors when the data cannot be loaded as is.
  3. You will a location confirmation - to make sure we selected the correct location for your activity - or maybe you need to create a new location.
  4. You will then see a preview of your activities. You still have a chance to Remove it from the load
  5. Once you confirm the load the activties will be available to users in WildPaths.
  6. If you find an issue with the load, you will have an option to ROLL it BACK and remove them wholesale from the system.
