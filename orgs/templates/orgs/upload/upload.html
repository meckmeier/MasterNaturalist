Upload Process

  1. Select your CSV file.
  2. Data will pass through an initial validation stage - you might see warnings for rows that have issues but can still load. Errors will prevent a row from loading.
  3. Next is a location confirmation - to make sure we selected the correct location for your activity.
  4. Then a preview of your activities. You can still remove an individual activity from the load
  5. Finally, PUBLISH the load to see the activity in WildPaths.
  6. If you find an issue with a load, you can undo and remove them wholesale from the system.
