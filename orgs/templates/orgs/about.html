Wild Paths - Wisconsin

Our mission: to connect nature-based organizations with Wisconsinites that care.

To gain access to all features of this site, you will need to register to create a login. Logging in will open all the menu options including search and activities.

To find opportunities

After you create a login, you can search for Activities (which include both volunteer opportunities and advanced trainings). You can also browse organizations, see a map of locations. Favorite organizations that you want to follow and later you can filter lists to just those organizations.

If you are a provider

If you are in an organizations you can use the Manage Organizations page to add activities or locations. If you are the owner of the org or location, you will see a pencil icon ( ) indicating that you can edit it. If you click that you will see a page where you can more details regarding your org/location. You will also see an add activity button. Use this add in new opportunities.
