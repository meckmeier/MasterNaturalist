About

This is a web page that shows Wisconsin Nature Based organizations that Master Naturalists can use to fulfill their ongoing training and volunteer requirements. Organizations can come here to add new training or volunteer opportunities.

To gain access to all features of this site, you will need to register to create a login. Logging in will open all the menu options including search and activities.

To find opportunities

After you create a login, you can see lists of Organizations, Locations or Activities. Each page has a filter option so you can filter.

If you are a provider

If you want to add new organizations or locations you can use the ADD menu choice. To add activities, you need to have already created an org or a location. If you are the owner of the org or location, you will see a pencil icon ( ) indicating that you can edit it. If you click that you will see a page where you can more details regarding your org/location. You will also see an add activity button. Use this add in new opportunities.
